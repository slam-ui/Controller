

## Тема
Реализация бэкенда для интернет-магазина с использованием Spring Boot и PostgreSQL.

## Сущности
1. **Product** (Товар) - имеет название, цену и количество на складе.
2. **Customer** (Клиент) - имеет имя и уникальный email.
3. **Order** (Заказ) - связывает Клиента и Товар, хранит статус и цену.

## Реализованные бизнес-операции
1. **Регистрация клиента** (с проверкой на дубликаты email).
2. **Оформление заказа** (Транзакционная операция: уменьшает остаток товара на складе).
3. **Оплата заказа** (Изменение статуса).
4. **Отмена заказа** (Транзакционная операция: возвращает товар на склад).
5. **Просмотр истории заказов** (Фильтрация заказов по клиенту).
